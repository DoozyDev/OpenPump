## About

This repository contains the complete OpenPump codebase and OpenJTE source code. It is a powerful [.NET](https://dotnet.microsoft.com) module that allows you to easily interact with the [PumpPortal API](https://pumpportal.fun) and build Solana-related applications.

- `OpenPump`
- `OpenPump.Abstractions`
- `OpenPump.Core`
- `OpenPump.Services.WebSocket`
